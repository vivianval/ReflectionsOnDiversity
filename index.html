Reflections on Diversity

A Real-time Virtual Mirror for Inclusive 3D Face Transformations

Presented by Paraskevi Valergaki, Antonis Argyros, Giorgos Giannakakis, and Anastasios Roussos

Abstract

This project presents a real-time interactive system for 3D face transformations that fosters diversity and inclusion. Using a combination of Generative Adversarial Networks (GANs) and 3D Morphable Models (3DMMs), we enable photorealistic modifications of faces across different ethnicities and genders while preserving individual identity.
